# 画像ファイルについて

このディレクトリに以下の画像ファイルを配置してください。

## 必要な画像ファイル

### ルート
- `logo.png` - サイトロゴ
- `firstimg.png` - ヒーローセクションのメイン画像

### team/ （チームメンバー）
- `seishi-ono.png`
- `kazuki-kasama.png`
- `yumi-misawa.png`
- `mikiko-tanifuji.png`
- `dieb-sae.png`

### partners/ （パートナー）
- `paul-walk.png`
- `luca-foppiano.png`
- `yuko-nagai.png`

### proposals/ （提案セクション）
- `pic1.png`
- `pic2.png`
- `pic3.png`

## 現在のWordPressサイトからの取得方法

以下のURLから画像をダウンロードできます：

```
https://cybermate.cloud/wp-content/uploads/2024/12/logo.png
https://cybermate.cloud/wp-content/uploads/2024/10/firstimg.png
https://cybermate.cloud/wp-content/uploads/2024/10/Seishi-Ono.png
...（以下同様）
```

## 推奨画像サイズ

- ロゴ: 幅200px程度
- チーム/パートナー写真: 200x200px（正方形）
- 提案セクション画像: 幅600px程度
- ヒーロー画像: 幅1200px程度
